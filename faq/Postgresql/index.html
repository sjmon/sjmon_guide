Postgresql

아래 FAQ 에 없는 문의사항은 담당자에게 직접 연락 바랍니다.

Windows 에서 설치 시, exe 파일 실행 오류

%TEMP% 위치의 bitrock_installer.log 를 확인하여 상세 로그를 확인해야 합니다.

ex) 2114 오류가 발생한 경우, Server 서비스가 disable 되어있어서 start 하고 다시 설치 시도하였습니다.

설치 중 Encoding 오류

OS의 Encoding 과 DB 설치 단계에 포함된 initDB -E UTF8 명령어와 충돌하여 오류가 발생할 수 있습니다. Postgresql 의 show server_encoding; 결과값이 LATIN1 으로 설정되어 Viewer 의 sqlMap 에 들어있는 한글 깨짐 오류 발생한 적 있어서 명시한 옵션입니다. OS 의 Encoding 이 euckr 인 경우, 해당 옵션 -E UTF8을 제외하고 설치해도 euckr 로 설정되어 정상적으로 사용 가능합니다. 그 외 Encoding 의 경우 담당자와 논의 및 추가 확인이 필요합니다.

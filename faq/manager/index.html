아래 FAQ 에 없는 문의사항은 담당자에게 직접 연락 바랍니다.

시작 시 pg_hba.conf entry for host 오류가 발생합니다.

db.properties 의 url 속 IP 설정을 localhost 로 수정한 후 재시작하여 정상 작동 여부를 확인합니다.

url=jdbc:postgresql://localhost:5432/sjmonv2repos

수정한 이후에도 해당 오류가 발생하거나 정상적으로 시작되지 않는 경우 설치 환경과 로그를 확인하여 상세 원인을 파악해야 합니다.

[WARN] io.netty.util.internal.MacAddressUtil 오류

맥 주소 값이 비어있는 경우 뜨는 로그입니다. 실행에 영향을 끼치지 않습니다.

(참고링크 https://github.com/netty/netty/commit/42c035982093d6e08201af9572ea43c7cf269adf)

./repo.sh start 시 Java Error

.profile에 path가 지정되어 있는지 확인해야 합니다.
